Project: Autonomous Robot Car

Overview:
This project implements an autonomous robot car capable of perceiving its environment,
planning a path, and executing motion commands.

Made by: 
Kevin Nguyen, Richard Nguyen

Repository Structure:
mobile-robot-maid/
├── frontend/
├── backend/
├── robot/

frontend/
Web interface for monitoring robot status and sending commands.

backend/
API server responsible for communication between the frontend and robot.

robot/
Software running on the Raspberry Pi and Arduino that handles perception,
planning, localization, and motor control.
